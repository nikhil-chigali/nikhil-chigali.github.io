# 💫 About Me
Data Scientist & ML Engineer building production AI systems — from LLM pipelines and multi-agent RAG to edge-deployed computer vision.

* 🔭 Current: Building real-time monitoring & anomaly detection for 12M+ devices at Verizon.
* 🤖 GenAI: Production RAG pipelines with hybrid retrieval, multi-agent systems, and LLMOps observability.
* 🚀 Recent: Published SWiM, an instance segmentation benchmark for spacecraft (arXiv:2507.10775).
* 🧠 Interests: LLM productionalization, Efficient Deep Learning (PEFT/LoRA), Onboard Computer Vision.
* 🏢 Alum: Ex-Microsoft ML & DS Consultant.
* 🎓 Education: M.S. Computer Science, Rice University (2024).
* 💬 Ask me about: RAG systems, LLMOps at scale, and model optimization for hardware constraints.
